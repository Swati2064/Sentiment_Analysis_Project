# 😊 Emotion Classifier using Machine Learning

An interactive **Emotion Classification** web application built using **Python**, **Scikit-learn**, and **Streamlit**. The application predicts the emotion expressed in a piece of text using a trained machine learning model.

---

## 📖 Project Overview

Emotion Classification is a Natural Language Processing (NLP) task that identifies the emotional state conveyed in text. This project trains a machine learning model on an emotion dataset and deploys it as a user-friendly Streamlit web application for real-time predictions.

---

## ✨ Features

* Predicts emotions from user-entered text
* Simple and interactive Streamlit interface
* Real-time emotion prediction
* Pre-trained machine learning model
* Easy to run locally
* Beginner-friendly project structure

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/Emotion_Classifier.git
cd Emotion_Classifier
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

If you don't have a requirements file:

```bash
pip install streamlit scikit-learn pandas numpy joblib
```

---

## ▶️ Run the Application

Start the Streamlit application using:

```bash
streamlit run app.py
```

The application will open automatically in your browser.

---

## 🧠 How It Works

1. User enters text into the Streamlit interface.
2. The trained machine learning model processes the text.
3. The model predicts the corresponding emotion.
4. The predicted emotion is displayed instantly.
